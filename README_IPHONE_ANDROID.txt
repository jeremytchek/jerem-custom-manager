Jerem Custom Manager - Version iPhone / Android

Installation iPhone :
1. Envoie ce ZIP sur iPhone via mail ou Google Drive.
2. Dézippe si nécessaire.
3. Ouvre index.html dans Safari.
4. Bouton Partager > Ajouter à l'écran d'accueil.

Installation Android :
1. Envoie ce ZIP sur Android via mail ou Google Drive.
2. Ouvre index.html dans Chrome.
3. Menu ⋮ > Ajouter à l'écran d'accueil.

Sauvegarde :
- Réglages > Exporter sauvegarde.
- Envoie le fichier JSON sur ta boîte mail ou Google Drive.
- Sur l'autre appareil : Importer sauvegarde.

Cette version est optimisée tactile : gros boutons, champs anti-zoom, menu horizontal mobile.
