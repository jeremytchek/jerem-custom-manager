INSTALLATION GRATUITE SUR IPHONE / ANDROID AVEC GITHUB PAGES

1) Créer un compte GitHub
- Va sur github.com
- Crée un compte gratuit

2) Créer un nouveau dépôt
- Clique sur New repository
- Nom du dépôt : jerem-custom-manager
- Mets-le en Public
- Clique Create repository

3) Envoyer les fichiers
- Dézippe ce dossier
- Envoie TOUS les fichiers du dossier dans le dépôt :
  index.html
  style.css
  app.js
  initial-data.js
  manifest.json
  sw.js
  README_GITHUB_PAGES.txt

4) Activer GitHub Pages
- Dans ton dépôt : Settings
- Puis Pages
- Source : Deploy from a branch
- Branch : main
- Folder : /root
- Save

5) Ouvrir sur iPhone
- Attends 1 à 2 minutes
- GitHub donnera un lien du style :
  https://TON-NOM.github.io/jerem-custom-manager/
- Ouvre ce lien avec Safari
- Bouton Partager
- Ajouter à l’écran d’accueil

6) Sauvegarde Google Drive / Mail
- Dans l’app : Réglages > Exporter sauvegarde
- Envoie le fichier JSON par mail ou Google Drive
- Sur l’autre appareil : Réglages > Importer sauvegarde

IMPORTANT
- L’application sera gratuite.
- Elle fonctionne sur iPhone, Android, Mac et Windows.
- Les données ne se synchronisent pas automatiquement.
- Pour rester à jour entre appareils : exporter/importer le fichier de sauvegarde JSON.
